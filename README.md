## PokeNext

Aplicação desenvolvida no framework Next.js no intuito de desenvolver minha habilidades

---
## Introdução 

Primeiro passo :

```bash
npm install
# or
yarn dev
```
para iniciar o projeto

```bash
npm run dev
# or
yarn dev
```

Depois basta abrir esse endereço no seu navegador[http://localhost:3000](http://localhost:3000)
